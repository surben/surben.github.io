2019-11-3看待金钱

Created2019-11-03|Updated2020-11-28
|Post View:

理财知识

知识来源，主要是B 站的up主：
第一个是：ztalk；
主要是从个人整体人生规划的角度，思考问题的方式上给出了纲领性的人生指导。
第二个是：狼爸爸的工作室；
用生动的动画，轻松深刻地展现了up主对于一些社会现象的个人认识。系统讲解了金融投资的核心概念知识。视频做的特别认真。作为八零后，关心九零后的想法，关心小孩子感兴趣的东西，这也是他的基本投资逻辑。作为完全他个人的理解认识。我感觉很赞同。
两个up主都是有着丰富投资经验的资深投资人。讲解逻辑和观点都很清晰。都有持续认知终身学习，都有积极健康的人生态度。都是有真才实学，有自己成熟思考和判断的海龟。
通过这个周末，整体翻看两位up主老师的视频。内心受到的震撼很深，看到了知识的力量，也见识了成熟人生的状态。感谢B站这个真实的平台，拉近了我与这么多优秀又愿意分享的大人物之间的距离。

有书

《机器人叛乱》，以形象比喻的方式，全景式地讲述了人类在“基因”和“模因”——生命传承和文化传承，两股力量的掌控下的抗拒与挣扎的生命全景。

这本书的视角很新颖，和我一直朦朦胧胧想要做的，通过视频动画的方式，形象展现了个人成长过程中内心的冲突问题的想法，有着不谋而合、异曲同工的妙处。

多读书，读好书。相信知识的力量。终身学习，不断自我进化。
